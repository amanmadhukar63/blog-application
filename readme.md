# 📝 Blog Application

A full-stack blog application where users can create, read, update, and delete blog posts. This project demonstrates core web development concepts including authentication, CRUD operations, and RESTful APIs.

---

## 🚀 Features

- User authentication (Signup / Login)
- Create, edit, and delete blog posts
- View all posts
- View individual post details
- Responsive UI
- RESTful API integration

---

## 🛠 Tech Stack

### Frontend
- React
- TypeScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
